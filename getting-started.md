# Ξεκινώντας με το Etherly Sync

Η εγκατάσταση ολοκληρώνεται σε **4 βήματα** και διαρκεί περίπου 10 λεπτά. Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- Λογαριασμό **Hosthub** με ένα τουλάχιστον ενεργό κατάλυμα
- Λογαριασμό **Elorus** με ένα τουλάχιστον organization
- Πρόσβαση στα API keys και των δύο υπηρεσιών

---

## Βήμα 1: Δημιουργία λογαριασμού

Μεταβείτε στο [app.etherly.app/auth/register](https://app.etherly.app/auth/register) και δημιουργήστε λογαριασμό. Θα λάβετε email επαλήθευσης — κάντε κλικ στον σύνδεσμο και θα μεταφερθείτε αυτόματα στον οδηγό εγκατάστασης.

<Info>
Ο λογαριασμός σας ξεκινά με **Δωρεάν Δοκιμαστική Περίοδος 30 ημερών** — απεριόριστες αποστολές, χωρίς κάρτα πληρωμής. Μετά τη λήξη απαιτούνται credits για την αποστολή παραστατικών.
</Info>

---

## Βήμα 2: Σύνδεση Hosthub

Στον οδηγό εγκατάστασης επιλέξτε το βήμα **«Σύνδεση Hosthub»**.

<Steps>
  <Step title="Βρείτε το API key σας στην Hosthub">
    Στη Hosthub πατήστε πάνω δεξιά, στο όνομα του προφίλ σας, και μετά **Ρυθμίσεις**. Στο κάτω μέρος της σελίδας θα βρείτε τις ρυθμίσεις API keys. Πατήστε «Δημιουργία νέου κλειδιού API», δώστε ένα όνομα και αποθηκεύστε. Αντιγράψτε το κλειδί που θα δημιουργηθεί.
  </Step>
  <Step title="Εισάγετε το key και δοκιμάστε τη σύνδεση">
    Επικολλήστε το key στο πεδίο **Hosthub API Key** και πατήστε **«Δοκιμή σύνδεσης»**.
  </Step>
  <Step title="Επιβεβαίωση και συνέχεια">
    Μετά την επιτυχή σύνδεση εμφανίζεται «Επιτυχής σύνδεση. Ο συγχρονισμός καταλυμάτων ξεκίνησε.» Πατήστε **«Συνέχεια»**.
  </Step>
</Steps>

---

## Βήμα 3: Σύνδεση Elorus

Επιλέξτε το βήμα **«Σύνδεση Elorus»**.

<Steps>
  <Step title="Βρείτε το API key σας στην Elorus">
    Στην Elorus πηγαίνετε **Ρυθμίσεις** (πάνω δεξιά, κλικ στο όνομά σας) → **Ρυθμίσεις API** → «Δημιουργία νέου κλειδιού». Αντιγράψτε το κλειδί που θα εμφανιστεί.
  </Step>
  <Step title="Εισάγετε το key και το Organization ID">
    Επικολλήστε το key στο πεδίο **Elorus API Key**. Για το **Organization ID**: στην Elorus πηγαίνετε Ρυθμίσεις → Οργανισμός — το ID εμφανίζεται στο URL της σελίδας.
  </Step>
  <Step title="Δοκιμή σύνδεσης και συνέχεια">
    Πατήστε **«Δοκιμή σύνδεσης»**. Μετά την επιτυχία εμφανίζεται «Επιτυχής σύνδεση. Έτοιμο για χρήση.» Πατήστε **«Συνέχεια»**.
  </Step>
</Steps>

<Warning>
Το API key της Elorus αποθηκεύεται **κρυπτογραφημένο** (AES-256-GCM) και δεν εμφανίζεται ποτέ σε plain text μετά την αποθήκευση.
</Warning>

---

## Βήμα 4: Ρύθμιση πρώτου καταλύματος

Πατήστε **«Μετάβαση στο Dashboard»** και μεταβείτε στα **Καταλύματα** από το sidebar. Κάθε κατάλυμα χρειάζεται τις παρακάτω ρυθμίσεις:

| Πεδίο | Τι είναι |
|-------|----------|
| **Προεπιλεγμένη Επαφή Αποδείξεων** | Ο λογαριασμός πελάτη στην Elorus για τις αποδείξεις |
| **Σειρά Τιμ.** | Η σειρά αρίθμησης τιμολογίων |
| **Κατηγορία** | Η κατηγορία για το Τέλος Κλιματικής Ανθεκτικότητας |
| **Οργανισμός** | Το Elorus organization έκδοσης παραστατικών |

<Tip>
Δείτε τον λεπτομερή [Οδηγό Ρύθμισης Καταλυμάτων](/guides/properties) για βήμα-προς-βήμα οδηγίες.
</Tip>

---

## Αποστολή πρώτου παραστατικού

Μόλις ρυθμίσετε ένα κατάλυμα, οι κρατήσεις του εμφανίζονται με κατάσταση **READY** (πράσινη ένδειξη).

<Steps>
  <Step title="Βρείτε μια κράτηση σε κατάσταση READY">
    Στο **Κρατήσεις**, οι έτοιμες κρατήσεις εμφανίζονται με πράσινη ένδειξη.
  </Step>
  <Step title="Πατήστε «Στείλε στην Elorus»">
    Κάντε κλικ στο κουμπί **«Στείλε στην Elorus»** δίπλα στην κράτηση, ή επιλέξτε πολλές για μαζική αποστολή.
  </Step>
  <Step title="Παρακολουθήστε την κατάσταση">
    Σε λίγα δευτερόλεπτα η κατάσταση αλλάζει σε **SENT**. Ο σύνδεσμος προς το παραστατικό στην Elorus εμφανίζεται αυτόματα.
  </Step>
</Steps>

<Check>
Το πρώτο σας παραστατικό εκδόθηκε επιτυχώς στην Elorus.
</Check>
