# Ξεκινώντας με το Etherly Sync

Η εγκατάσταση ολοκληρώνεται σε **4 βήματα** και διαρκεί περίπου 10 λεπτά. Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- Λογαριασμό **Hosthub** με ένα τουλάχιστον ενεργό ακίνητο
- Λογαριασμό **Elorus** με ένα τουλάχιστον organization
- Πρόσβαση στα API keys και των δύο υπηρεσιών

---

## Βήμα 1: Δημιουργία λογαριασμού

Μεταβείτε στο [app.etherly.io/auth/register](https://app.etherly.io/auth/register) και δημιουργήστε λογαριασμό. Θα λάβετε email επαλήθευσης — κάντε κλικ στον σύνδεσμο και θα μεταφερθείτε αυτόματα στον οδηγό εγκατάστασης.

<Info>
Ο λογαριασμός σας ξεκινά με **30 ημέρες δωρεάν δοκιμή** — απεριόριστες αποστολές, χωρίς κάρτα πληρωμής.
</Info>

---

## Βήμα 2: Σύνδεση Hosthub

<Steps>
  <Step title="Βρείτε το API key σας στο Hosthub">
    Συνδεθείτε στο Hosthub → **Settings** → **API** → αντιγράψτε το **Personal API Token**.
  </Step>
  <Step title="Εισάγετε το key στο Etherly Sync">
    Στον οδηγό εγκατάστασης, επιλέξτε **"Connect Hosthub"**, επικολλήστε το key και πατήστε **Validate**.
  </Step>
  <Step title="Επιβεβαίωση σύνδεσης">
    Το σύστημα θα εντοπίσει αυτόματα τα ακίνητά σας και θα εμφανίσει το πλήθος τους. Αν δείτε σφάλμα, βεβαιωθείτε ότι αντιγράψατε ολόκληρο το token χωρίς κενά.
  </Step>
</Steps>

---

## Βήμα 3: Σύνδεση Elorus

<Steps>
  <Step title="Βρείτε το API key σας στο Elorus">
    Συνδεθείτε στο Elorus → **Ρυθμίσεις** → **Ενσωματώσεις** → **API Keys** → δημιουργήστε νέο key ή αντιγράψτε υπάρχον.
  </Step>
  <Step title="Εισάγετε το key στο Etherly Sync">
    Επιλέξτε **"Connect Elorus"**, επικολλήστε το key και πατήστε **Validate**.
  </Step>
  <Step title="Αυτόματη ανίχνευση ρυθμίσεων">
    Το Etherly Sync ανακτά αυτόματα από το Elorus:
    - Τα organizations σας
    - Τον ΦΠΑ 13% (ή τον πλησιέστερο διαθέσιμο)
    - Τους τύπους παραστατικών (τιμολόγιο, απόδειξη)
  </Step>
</Steps>

<Warning>
Το API key του Elorus αποθηκεύεται **κρυπτογραφημένο** (AES-256-GCM) και δεν εμφανίζεται ποτέ σε plain text μετά την αποθήκευση.
</Warning>

---

## Βήμα 4: Ρύθμιση πρώτου ακινήτου

Μετά τη σύνδεση, τα ακίνητά σας εμφανίζονται στο dashboard. Για να ξεκινήσει η τιμολόγηση, κάθε ακίνητο χρειάζεται τέσσερις ρυθμίσεις:

| Ρύθμιση | Τι είναι |
|---------|----------|
| **Elorus Contact** | Ο λογαριασμός πελάτη στο Elorus (π.χ. "Ιδιώτης Πελάτης") |
| **Invoice Sequence** | Η σειρά αρίθμησης παραστατικών (π.χ. "ΤΔΑ 2025") |
| **Κατηγορία Τέλους** | Η κατηγορία για το Τέλος Κλιματικής Ανθεκτικότητας |
| **Elorus Organization** | Το organization υπό το οποίο εκδίδονται τα παραστατικά |

<Tip>
Δείτε τον λεπτομερή [Οδηγό Ρύθμισης Ακινήτων](/guides/properties) για βήμα-προς-βήμα οδηγίες.
</Tip>

---

## Αποστολή πρώτου τιμολογίου

Μόλις ρυθμίσετε ένα ακίνητο, τα checkouts του εμφανίζονται με κατάσταση **READY** (πράσινη ένδειξη).

<Steps>
  <Step title="Βρείτε ένα checkout σε κατάσταση READY">
    Στο κύριο dashboard, τα έτοιμα checkouts εμφανίζονται με πράσινη ένδειξη.
  </Step>
  <Step title="Πατήστε «Send»">
    Κάντε κλικ στο κουμπί **Send** δίπλα στο checkout ή επιλέξτε πολλά για μαζική αποστολή.
  </Step>
  <Step title="Παρακολουθήστε την κατάσταση">
    Σε λίγα δευτερόλεπτα η κατάσταση αλλάζει σε **SENT**. Ο σύνδεσμος προς το παραστατικό στο Elorus εμφανίζεται αυτόματα.
  </Step>
</Steps>

<Check>
Συγχαρητήρια! Το πρώτο σας τιμολόγιο εκδόθηκε αυτόματα στο Elorus και υποβλήθηκε στο myDATA.
</Check>
