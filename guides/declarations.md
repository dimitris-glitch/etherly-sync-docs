# Δηλώσεις Βραχυχρόνιας Μίσθωσης

Κάθε τιμολογημένη κράτηση πρέπει να δηλωθεί στην ΑΑΔΕ μέσω του **AADE-Connect**. Το Etherly Sync αναλαμβάνει αυτόματα την αποστολή για λογαριασμό σας.

Οι δηλώσεις βρίσκονται στο **Δηλώσεις** στο sidebar — εμφανίζονται μόνο κρατήσεις που έχουν ήδη τιμολογηθεί και έχουν αντιστοίχηση ΑΑΔΕ ακινήτου.

## Καταστάσεις δήλωσης

| Κατάσταση | Τι σημαίνει |
|-----------|-------------|
| Μη υποβληθείσα | Η δήλωση δεν έχει σταλεί ακόμα |
| Στάλθηκε… | Αποστολή σε εξέλιξη |
| Sent | Προσωρινή δήλωση (draft) στάλθηκε στο AADE-Connect |
| Draft | Η δήλωση βρίσκεται σε προσωρινή κατάσταση στο AADE-Connect |
| Υποβλήθηκε | Οριστική υποβολή — δεν απαιτείται καμία ενέργεια |
| Αποτυχία | Η αποστολή απέτυχε — ελέγξτε το μήνυμα σφάλματος |
| Παραλείφθηκε | Εξαιρέθηκε χειροκίνητα από τη δήλωση |

## Χειροκίνητη υποβολή

<Steps>
  <Step title="Επιλέξτε κρατήσεις">
    Στη σελίδα **Δηλώσεις**, επιλέξτε μία ή περισσότερες κρατήσεις με checkbox.
  </Step>
  <Step title="Επιλέξτε τύπο δήλωσης">
    Χρησιμοποιήστε το κουμπί **«Αποστολή Επιλεγμένων»** και επιλέξτε:
    - **Οριστική** — οριστική υποβολή στην ΑΑΔΕ
    - **Προσωρινή** — draft για μελλοντική οριστικοποίηση
  </Step>
  <Step title="Παρακολουθήστε την κατάσταση">
    Η κατάσταση ενημερώνεται αυτόματα. Σε αποτυχία εμφανίζεται το μήνυμα σφάλματος inline.
  </Step>
</Steps>

## Αυτόματη οριστική δήλωση

Ενεργοποιήστε την **αυτόματη οριστική δήλωση** από τη σελίδα Δηλώσεις (toggle πάνω από τον πίνακα). Με την ενεργοποίηση, νέες τιμολογημένες κρατήσεις δηλώνονται αυτόματα κάθε μέρα.

<Note>
Η αυτόματη δήλωση εφαρμόζεται μόνο σε κρατήσεις που τιμολογούνται **μετά** την ενεργοποίησή της — δεν αναδρομολογεί παλαιότερες κρατήσεις.
</Note>

## Ταυτοποίηση επισκέπτη

Κάθε δήλωση μπορεί να συμπεριλαμβάνει τα στοιχεία ταυτοποίησης του επισκέπτη. Κάντε κλικ στο **εικονίδιο μολυβιού** στη στήλη «Ταυτοποίηση» για να επεξεργαστείτε.

| Τύπος | Πότε χρησιμοποιείται |
|-------|----------------------|
| Αστυνομική Ταυτότητα | Έλληνας πολίτης με ΑΔΤ |
| Διαβατήριο | Αλλοδαπός επισκέπτης |
| ΑΦΜ | Έλληνας με ΑΦΜ (αντί ΑΔΤ) |
| Reservation ID | ID κράτησης από κανάλι — αν δεν υπάρχουν άλλα στοιχεία |

<Tip>
Αν η κράτηση έχει ήδη Reservation ID από το Hosthub, το πεδίο προσυμπληρώνεται αυτόματα.
</Tip>

## Αντιστοίχιση πλατφόρμας

Το Etherly Sync αντιστοιχίζει αυτόματα το κανάλι κράτησης στον κωδικό πλατφόρμας που απαιτεί η ΑΑΔΕ. Η στήλη **Κανάλι** στον πίνακα Δηλώσεων δείχνει την αναγνωρισμένη πλατφόρμα — όχι το ακατέργαστο όνομα από το Hosthub.

| Κανάλι κράτησης | Εμφάνιση στη στήλη | Κωδικός ΑΑΔΕ |
|-----------------|--------------------|--------------|
| Airbnb, Airbnb Plus, Airbnb (Greece) κ.λπ. | Airbnb | `AIRBNB` |
| Booking.com, Booking.com for … κ.λπ. | Booking.com | `BOOKING_COM` |
| Clickstay | Clickstay | `CLICKSTAY` |
| HomeAway, VRBO, Expedia κ.λπ. | HomeAway / VRBO | `HOMEAWAY` |
| Homestay | Homestay | `HOMESTAY` |
| Luxury Retreats | Luxury Retreats | `LUXURY_RETREATS` |
| Only Apartments | Only Apartments | `ONLY_APARTMENTS` |
| TripAdvisor, Holiday Lettings κ.λπ. | TripAdvisor | `TRIPADVISOR_RENTALS` |
| Τηλεφωνική / Direct / κενό | Εκτός πλατφόρμας | `OTHER_DIGITAL_PLATFORMS` · `"Εκτός πλατφόρμας"` |
| Οποιοδήποτε άλλο κανάλι | (ακατέργαστο όνομα) | `OTHER_DIGITAL_PLATFORMS` · το όνομα του καναλιού |

Το σύστημα αναγνωρίζει και **σύνθετα ονόματα sub-channel** του Hosthub (π.χ. `"Booking.com for Ariadnis + Collection"`, `"Airbnb Plus"`) μέσω substring matching — η αντιστοίχιση γίνεται σωστά ακόμα και αν το όνομα δεν είναι ακριβές.

<Warning>
Οι **απευθείας κρατήσεις** (τηλεφωνικές, email, δικό σας site) στέλνονται πάντα με `platform_name: "Εκτός πλατφόρμας"` — αυτό επιβάλλει η ΑΑΔΕ. Το σύστημα το χειρίζεται αυτόματα.
</Warning>

## Ιστορικό δηλώσεων

Η καρτέλα **Ιστορικό** στη σελίδα Δηλώσεων δείχνει κρατήσεις που έχουν ήδη υποβληθεί ή παραλειφθεί. Για αυτές τις κρατήσεις δεν επιτρέπεται επεξεργασία στοιχείων ταυτοποίησης.
