# Δηλώσεις Βραχυχρόνιας Μίσθωσης

Αν έχετε ενεργοποιήσει τη λειτουργία δηλώσεων ΑΑΔΕ, οι επιλέξιμες τιμολογημένες κρατήσεις μπορούν να αποσταλούν μέσω **AADE-Connect** από το Etherly Sync — χειροκίνητα ή αυτόματα.

Οι δηλώσεις βρίσκονται στο **Δηλώσεις** στο sidebar — εμφανίζονται μόνο κρατήσεις που έχουν ήδη τιμολογηθεί και έχουν αντιστοίχηση ΑΑΔΕ ακινήτου.

## Καταστάσεις δήλωσης

| Κατάσταση | Τι σημαίνει |
|-----------|-------------|
| Μη υποβληθείσα | Η δήλωση δεν έχει σταλεί ακόμα |
| Στάλθηκε… | Αποστολή σε εξέλιξη |
| Sent | Προσωρινή δήλωση (draft) στάλθηκε στο AADE-Connect |
| Draft | Η δήλωση βρίσκεται σε προσωρινή κατάσταση στο AADE-Connect |
| Υποβλήθηκε | Οριστική υποβολή — δεν απαιτείται καμία ενέργεια |
| Αποτυχία | Η αποστολή απέτυχε — ελέγξτε το μήνυμα σφάλματος |
| Παραλείφθηκε | Εξαιρέθηκε χειροκίνητα από τη δήλωση |

## Χειροκίνητη υποβολή

<Steps>
  <Step title="Επιλέξτε κρατήσεις">
    Στη σελίδα **Δηλώσεις**, επιλέξτε μία ή περισσότερες κρατήσεις με checkbox.
  </Step>
  <Step title="Επιλέξτε τύπο δήλωσης">
    Χρησιμοποιήστε το κουμπί **«Αποστολή Επιλεγμένων»** και επιλέξτε:
    - **Οριστική** — οριστική υποβολή στην ΑΑΔΕ
    - **Προσωρινή** — draft για μελλοντική οριστικοποίηση
  </Step>
  <Step title="Παρακολουθήστε την κατάσταση">
    Η κατάσταση ενημερώνεται αυτόματα. Σε αποτυχία εμφανίζεται το μήνυμα σφάλματος inline.
  </Step>
</Steps>

## Αυτόματη οριστική δήλωση

Ενεργοποιήστε την **αυτόματη οριστική δήλωση** από τη σελίδα Δηλώσεις (toggle πάνω από τον πίνακα). Με την ενεργοποίηση, νέες τιμολογημένες κρατήσεις δηλώνονται αυτόματα κάθε μέρα.

<Note>
Η αυτόματη δήλωση εφαρμόζεται μόνο σε κρατήσεις που τιμολογούνται **μετά** την ενεργοποίησή της — δεν αναδρομολογεί παλαιότερες κρατήσεις.
</Note>

## Ταυτοποίηση επισκέπτη

Κάθε δήλωση μπορεί να συμπεριλαμβάνει τα στοιχεία ταυτοποίησης του επισκέπτη. Κάντε κλικ στο **εικονίδιο μολυβιού** στη στήλη «Ταυτοποίηση» για να επεξεργαστείτε.

| Τύπος | Πότε χρησιμοποιείται |
|-------|----------------------|
| ΑΦΜ | Έλληνας επισκέπτης — μόνο ενεργό ΑΦΜ γίνεται αποδεκτό |
| Διαβατήριο | Αλλοδαπός επισκέπτης με διαβατήριο |
| Αστυνομική Ταυτότητα | Αλλοδαπός επισκέπτης με ταυτότητα (π.χ. χώρα ΕΕ) |
| Reservation ID | Fallback — αν δεν υπάρχει ΑΦΜ ή ταυτοποιητικό έγγραφο |

<Tip>
Αν η κράτηση έχει ήδη Reservation ID από το κανάλι κρατήσεων, το πεδίο προσυμπληρώνεται αυτόματα.
</Tip>

## Αντιστοίχιση πλατφόρμας

Το Etherly Sync αντιστοιχίζει αυτόματα το κανάλι κράτησης στον κωδικό πλατφόρμας που απαιτεί η ΑΑΔΕ. Η στήλη **Κανάλι** στον πίνακα Δηλώσεων δείχνει την αναγνωρισμένη πλατφόρμα — όχι το ακατέργαστο όνομα από το κανάλι.

| Κανάλι κράτησης | Εμφάνιση στη στήλη |
|-----------------|--------------------|
| Airbnb, Airbnb Plus, Airbnb (Greece) κ.λπ. | Airbnb |
| Booking.com, Booking.com for … κ.λπ. | Booking.com |
| Clickstay | Clickstay |
| HomeAway, VRBO, Expedia κ.λπ. | HomeAway / VRBO |
| Homestay | Homestay |
| Luxury Retreats | Luxury Retreats |
| Only Apartments | Only Apartments |
| TripAdvisor, Holiday Lettings κ.λπ. | TripAdvisor |
| Τηλεφωνική / Direct / κενό | Εκτός πλατφόρμας |
| Οποιοδήποτε άλλο κανάλι | (ακατέργαστο όνομα) |

Το σύστημα αναγνωρίζει και **σύνθετα ονόματα sub-channel** (π.χ. `"Booking.com for Athens"`, `"Airbnb Plus"`) μέσω substring matching — η αντιστοίχιση γίνεται σωστά ακόμα και αν το όνομα δεν είναι ακριβές.

<Warning>
Οι **απευθείας κρατήσεις** (τηλεφωνικές, email, δικό σας site) στέλνονται πάντα ως «Εκτός πλατφόρμας» — αυτό επιβάλλει η ΑΑΔΕ. Το σύστημα το χειρίζεται αυτόματα.
</Warning>

## Τρόπος πληρωμής μισθώματος

Κάθε δήλωση στέλνεται στην ΑΑΔΕ με πληροφορία για **πού πληρώνει το κανάλι** τον ιδιοκτήτη. Οι επιτρεπόμενες τιμές είναι: Τράπεζα στην Ελλάδα, Τράπεζα στο Εξωτερικό, Μετρητά, Άλλο (μέσω τρίτων, voucher κλπ).

### Προεπιλογή ανά κανάλι

Στις **Ρυθμίσεις → AADE Connect** μπορείτε να ορίσετε προεπιλεγμένο τρόπο πληρωμής για κάθε κανάλι κρατήσεων. Η αλλαγή επηρεάζει αυτόματα **όλες τις μελλοντικές υποβολές** εκείνου του καναλιού — αρκεί να μην υπάρχει χειροκίνητη παράκαμψη ανά κράτηση.

### Παράκαμψη ανά κράτηση

Στη στήλη **Τρόπος Πληρωμής** της σελίδας Δηλώσεων, το εικονίδιο ✏️ επιτρέπει την αλλαγή τρόπου πληρωμής για μία συγκεκριμένη κράτηση. Η παράκαμψη **υπερισχύει** πάντα της προεπιλογής καναλιού, και εμφανίζεται με κανονική απόχρωση ενώ οι τιμές από προεπιλογή εμφανίζονται απαλά.

Η παράκαμψη επιτρέπεται μόνο για δηλώσεις σε κατάσταση **Μη υποβληθείσα (draft)** ή **Αποτυχία χωρίς checkpoint**. Δεν επιτρέπεται για δηλώσεις που έχουν ήδη υποβληθεί στην ΑΑΔΕ.

## Ιστορικό δηλώσεων

Η καρτέλα **Ιστορικό** στη σελίδα Δηλώσεων δείχνει κρατήσεις που έχουν ήδη υποβληθεί ή παραλειφθεί. Για αυτές τις κρατήσεις δεν επιτρέπεται επεξεργασία στοιχείων ταυτοποίησης.
