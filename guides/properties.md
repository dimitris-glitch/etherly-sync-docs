# Ρύθμιση Καταλυμάτων

Εδώ ορίζετε για κάθε κατάλυμα όλες τις πληροφορίες που χρειάζεται η εφαρμογή για να εκδίδει παραστατικά σωστά: στοιχεία εταιρείας, τύπους παραστατικών, φόρους, περιγραφές υπηρεσιών κ.ά. Η ρύθμιση γίνεται **μία φορά** — στη συνέχεια η εφαρμογή μπορεί να λειτουργεί αυτόματα για κάθε νέα κράτηση του καταλύματος.

Παράλληλα, παραμένει πάντα η δυνατότητα χειροκίνητης διαχείρισης για κρατήσεις που θέλετε να χειριστείτε διαφορετικά — π.χ. έκδοση τιμολογίου αντί απόδειξης, αποστολή σε διαφορετικό χρόνο, ή εξαίρεση από την αυτόματη τιμολόγηση.

Σε κάθε κατάλυμα πρέπει να έχουν περαστεί όλες οι ρυθμίσεις για να μπορέσει να γίνει η έκδοση παραστατικών για τις κρατήσεις του. Χωρίς ρύθμιση, οι κρατήσεις του εμφανίζονται σε κατάσταση `NEEDS_SETUP`.

## Απαιτούμενα πεδία

### 1. Προεπιλεγμένη Επαφή Αποδείξεων

Ο λογαριασμός στην Elorus που εμφανίζεται ως «πελάτης» στις αποδείξεις — συνήθως ένας λιανικός λογαριασμός.

**Στην Elorus:**
1. Μεταβείτε στην Elorus → **Επαφές**
2. Βρείτε ή δημιουργήστε έναν πελάτη λιανικής

**Στα Καταλύματα → Επαφή Αποδ.:**
Κάντε κλικ στο πεδίο και πληκτρολογήστε το **όνομα** της επαφής (από 3+ χαρακτήρες) για αναζήτηση.

### 2. Σειρά Τιμ. (Σειρά Τιμολογίων)

Η σειρά αρίθμησης για τα τιμολόγια του καταλύματος. Αντίστοιχα, το πεδίο **Σειρά Απ.** (Σειρά Αποδείξεων) αφορά τις αποδείξεις.

<Warning>
Αν η σειρά δεν έχει ενεργοποιηθεί στην Elorus ή δεν επιτρέπει τον τύπο παραστατικού που χρησιμοποιείτε, η αποστολή θα αποτύχει. Επαληθεύστε στην Elorus → Ρυθμίσεις → Τύποι Εγγράφων → {επιλέξτε τον σωστό τύπο} → Σειρές αρίθμησης.
</Warning>

### 3. Κατηγορία

Η κατηγορία του καταλύματος για τον υπολογισμό του Τέλους Κλιματικής Ανθεκτικότητας. Επιλογές:

- **Standard** — τυπικό κατάλυμα
- **>80τμ** — σπίτια άνω των 80 τ.μ.

<Tip>
Δείτε τον [Οδηγό Τέλους Κλιματικής Ανθεκτικότητας](/guides/climate-fee) για τον τρόπο υπολογισμού και παραδείγματα.
</Tip>

### 4. Οργανισμός

Το Elorus organization υπό το οποίο εκδίδονται τα παραστατικά. Αν διαχειρίζεστε καταλύματα υπό διαφορετικά ΑΦΜ, μπορείτε να επιλέξετε διαφορετικό **Οργανισμός** ανά κατάλυμα.

## Βήμα-προς-βήμα ρύθμιση

<Steps>
  <Step title="Μεταβείτε στα Καταλύματα">
    Επιλέξτε **Καταλύματα** από το sidebar. Εναλλακτικά, κλικ στο εικονίδιο ρυθμίσεων δίπλα σε μια κράτηση `NEEDS_SETUP`.
  </Step>
  <Step title="Επιλέξτε κατάλυμα">
    Θα δείτε τη λίστα όλων των καταλυμάτων που έχουν εντοπιστεί από τη Hosthub. Επιλέξτε αυτό που θέλετε να ρυθμίσετε.
  </Step>
  <Step title="Συμπληρώστε τα πεδία">
    Επιλέξτε τιμές για **Προεπιλεγμένη Επαφή Αποδείξεων**, **Σειρά Τιμ.**, **Κατηγορία** και **Οργανισμός** από τις αντίστοιχες λίστες.
  </Step>
  <Step title="Αποθηκεύστε">
    Πατήστε **«Αποθήκευση»**. Οι κρατήσεις του καταλύματος αλλάζουν από `NEEDS_SETUP` σε `READY` στον επόμενο συγχρονισμό.
  </Step>
</Steps>

## Ενεργοποίηση / Απενεργοποίηση καταλύματος

Κάθε κατάλυμα μπορεί να ενεργοποιηθεί ή να απενεργοποιηθεί με τα κουμπιά **«Ενεργοποίηση καταλύματος»** / **«Απενεργοποίηση καταλύματος»**.

| Κατάσταση | Τιμολόγηση | Συγχρονισμός |
|-----------|-----------|--------------|
| **On** | ✅ Κανονική | ✅ Συνεχής |
| **Off** | ❌ Ανασταλμένη | ✅ Συνεχής |

<Note>
Το **Off** δεν σταματά τον συγχρονισμό. Τα δεδομένα παραμένουν ενημερωμένα — μόνο η τιμολόγηση αναστέλλεται. Ιδανικό για καταλύματα υπό ανακαίνιση ή εκτός εποχής.
</Note>

## Πολλαπλοί οργανισμοί Elorus

Αν διαχειρίζεστε καταλύματα υπό **διαφορετικά ΑΦΜ**, προσθέστε πολλαπλές συνδέσεις Elorus. Για παράδειγμα, αν έχετε κάποια ακίνητα υπό ατομική επιχείρηση και άλλα υπό εταιρεία, δημιουργήστε ξεχωριστή σύνδεση για κάθε ΑΦΜ — και αντιστοιχίστε κάθε κατάλυμα στον σωστό οργανισμό.

<Steps>
  <Step title="Προσθήκη νέας σύνδεσης">
    **Ρυθμίσεις** → **Διασυνδέσεις** → **«Νέα Σύνδεση»**
  </Step>
  <Step title="Εισάγετε το νέο Elorus API key">
    Κάθε σύνδεση απαιτεί ξεχωριστό API key. Δώστε έναν **Τίτλο Σύνδεσης** για να την αναγνωρίζετε εύκολα (π.χ. «My Elorus»).
  </Step>
  <Step title="Αντιστοιχίστε καταλύματα">
    Επιστρέψτε στα **Καταλύματα** και ορίστε το σωστό **Οργανισμός** για κάθε κατάλυμα.
  </Step>
</Steps>

## Copy / Paste ρυθμίσεων

Αν έχετε πολλά καταλύματα με τις ίδιες ρυθμίσεις, χρησιμοποιήστε **«Αντιγραφή ρυθμίσεων»** σε ένα κατάλυμα και **«Επικόλληση ρυθμίσεων»** στα υπόλοιπα για γρήγορη ρύθμιση.
