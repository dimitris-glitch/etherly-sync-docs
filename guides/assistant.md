# Etherly Assistant

Ο **Etherly Assistant** είναι ο ενσωματωμένος βοηθός AI στο Etherly Sync. Κλικ στο κουμπί κάτω δεξιά σε οποιαδήποτε σελίδα για να τον ανοίξετε (Escape για κλείσιμο).

## Τι μπορεί να κάνει

- Απαντά σε ερωτήσεις για τη λειτουργία της εφαρμογής, τον ΦΠΑ, το myDATA και την Ειδική Εισφορά Κλιματικής Ανθεκτικότητας
- Εντοπίζει ελλείψεις στη ρύθμιση και σας καθοδηγεί βήμα βήμα
- Αναθέτει οργανισμούς, κλίμακες ΦΠΑ και εισφορές σε ακίνητα για λογαριασμό σας (μετά από επιβεβαίωση)
- Ελέγχει την κατάσταση συγχρονισμού — worker, τελευταίος συγχρονισμός, κολλημένες ή αποτυχημένες εργασίες
- Αναζητά κράτηση βάσει ονόματος επισκέπτη ή ID και εξηγεί γιατί ένα παραστατικό δεν εκδόθηκε
- Πλοηγείται αυτόματα στη σχετική σελίδα ρυθμίσεων όταν χρειάζεται

## Τι δεν μπορεί να κάνει

- Να εκδώσει, ακυρώσει ή επανεκδώσει παραστατικά απευθείας — η έκδοση γίνεται αυτόματα όταν μια κράτηση είναι έτοιμη
- Να εμφανίσει email, τηλέφωνο ή άλλα προσωπικά στοιχεία επισκέπτη — μόνο όνομα, ακίνητο, ημερομηνίες και κατάσταση

## Ιστορικό συνομιλιών

Κάθε συνομιλία αποθηκεύεται αυτόματα. Κλικ στο **εικονίδιο ρολογιού** πάνω δεξιά στο panel για να δείτε παλιές συνομιλίες ομαδοποιημένες κατά ημερομηνία. Κλικ σε οποιαδήποτε για να τη συνεχίσετε.

Για νέα συνομιλία, χρησιμοποιήστε το **κουμπί επαναφοράς** στην κεφαλίδα ή το κουμπί **«Νέα Συνομιλία»** στην προβολή ιστορικού.

## Ασφάλεια και απόρρητο

Κάθε χρήστης βλέπει μόνο τις δικές του συνομιλίες — ακόμα και αν υπάρχουν πολλαπλοί χρήστες στον ίδιο λογαριασμό.
