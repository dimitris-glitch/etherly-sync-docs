## Τι είναι το Etherly Insights

Το Etherly Insights επιτρέπει σε εργαλεία AI (Claude, ChatGPT, Gemini και άλλα που υποστηρίζουν MCP ή OpenAPI) να διαβάζουν τα δεδομένα κρατήσεων και εσόδων σας, ώστε να μπορείτε να κάνετε ερωτήσεις όπως «Πόσα έσοδα είχα τον προηγούμενο μήνα;» ή «Βρες την κράτηση του Γιώργου» απευθείας στο εργαλείο AI της επιλογής σας.

<Note>
Το Etherly Insights είναι **αυστηρά μόνο για ανάγνωση**. Κανένα εργαλείο AI δεν μπορεί να δημιουργήσει, να τροποποιήσει ή να διαγράψει δεδομένα μέσω αυτής της σύνδεσης.
</Note>

---

## Απόρρητο επισκεπτών

Τα στοιχεία επισκεπτών **δεν** μοιράζονται ποτέ μέσω του Etherly Insights: όνομα, email, τηλέφωνο και αριθμός ταυτοποίησης δεν εμφανίζονται ποτέ στα αποτελέσματα. Επιτρέπονται μόνο μη-προσωποποιήσιμα στοιχεία (διανυκτερεύσεις, ποσά, κανάλι κράτησης, χώρα προέλευσης, εθνικότητα βάσει κωδικού χώρας τηλεφώνου).

## Ενεργοποίηση

<Steps>
  <Step title="Μεταβείτε στις ρυθμίσεις">
    Πηγαίνετε στις **Ρυθμίσεις → Διασυνδέσεις** και βρείτε την κάρτα **Etherly Insights** στην
    ενότητα «Apps». Πατήστε **«Ενεργοποίηση»** — ανοίγει ένα αναδυόμενο παράθυρο συναίνεσης.
  </Step>
  <Step title="Ενεργοποιήστε τη λειτουργία">
    Διαβάστε τη σύντομη περιγραφή συναίνεσης στο παράθυρο και πατήστε **«Ενεργοποίηση»**. Μόλις
    ενεργοποιηθεί, η καρτέλα **«Etherly Insights»** εμφανίζεται μόνιμα στο μενού Ρυθμίσεων και
    μπορείτε να πατήσετε **«Ρύθμιση»** στο ίδιο παράθυρο για να συνεχίσετε.
  </Step>
  <Step title="Δημιουργήστε ένα κλειδί API">
    Στην καρτέλα **Etherly Insights**, πατήστε **«Νέο key»**, δώστε έναν τίτλο (π.χ. «Claude
    Desktop») και πατήστε **«Δημιουργία»**.

    <Warning>
    Το κλειδί εμφανίζεται **μία μόνο φορά**. Αντιγράψτε το αμέσως — δεν θα μπορείτε να το δείτε ξανά ολόκληρο.
    </Warning>
  </Step>
</Steps>

## Σύνδεση με εργαλείο AI

Στην ίδια σελίδα θα βρείτε δύο endpoints:

| Endpoint | Χρήση |
|---|---|
| **MCP endpoint** (`/api/insights/mcp`) | Για εργαλεία που υποστηρίζουν MCP (π.χ. Claude Desktop, custom connectors) |
| **OpenAPI document** (`/api/insights/v1/openapi.json`) | Για εργαλεία που συνδέονται μέσω REST/OpenAPI (π.χ. ChatGPT custom actions) |

### Σύνδεση μέσω Claude.ai (OAuth) — χωρίς API key

Η λειτουργία **«Add custom connector»** του Claude.ai υποστηρίζει μόνο OAuth, όχι απευθείας API key. Για να συνδεθείτε:

<Steps>
  <Step title="Προσθήκη connector">
    Στο Claude.ai, πηγαίνετε στις **Settings → Connectors → Add custom connector** και επικολλήστε
    το URL του **MCP endpoint** (`https://<το-domain-σας>/api/insights/mcp`). Δεν χρειάζεται
    κλειδί — αφήστε κενά τυχόν πεδία «OAuth Client ID / Secret».
  </Step>
  <Step title="Σύνδεση και έγκριση">
    Το Claude ανοίγει μια οθόνη σύνδεσης και συναίνεσης. Συνδεθείτε με τον συνήθη λογαριασμό
    σας στο hosthub-elorus και πατήστε **«Allow»** για να δώσετε πρόσβαση.
  </Step>
  <Step title="Ολοκληρώθηκε">
    Η σύνδεση εμφανίζεται στις **Ρυθμίσεις → Etherly Insights** μαζί με τυχόν κλειδιά API που
    έχετε δημιουργήσει χειροκίνητα, με ένδειξη **«Συνδεδεμένη εφαρμογή»**, ημερομηνία
    **«Συνδέθηκε στις»** και δικό της κουμπί αποσύνδεσης.
  </Step>
</Steps>

<Note>
Αν το Etherly Insights δεν ήταν ήδη ενεργοποιημένο, η έγκριση της σύνδεσης με το Claude.ai το
ενεργοποιεί αυτόματα — δεν χρειάζεται ξεχωριστό βήμα χειροκίνητης ενεργοποίησης.
</Note>

### Σύνδεση με οτιδήποτε άλλο (χειροκίνητο κλειδί API)

Για τοπικές ρυθμίσεις με βάση το `mcp-remote`, ή για εργαλεία REST/OpenAPI όπως τα custom
actions του ChatGPT, δημιουργήστε ένα κλειδί API όπως περιγράφεται παραπάνω και
χρησιμοποιήστε το ως **Bearer token** στο header `Authorization` όταν ρυθμίζετε τη σύνδεση.

## Διαθέσιμα εργαλεία (Phase 1)

| Εργαλείο | Περιγραφή |
|---|---|
| **business-overview** | Συνολικές κρατήσεις, διανυκτερεύσεις, έσοδα και ανάλυση ανά κανάλι για ένα εύρος ημερομηνιών |
| **booking-search** | Αναζήτηση κράτησης βάσει ονόματος επισκέπτη ή κωδικού κράτησης |

Περισσότερα εργαλεία θα προστεθούν σταδιακά.

---

## Όρια χρήσης

- **Rate limit**: έως 20 κλήσεις/λεπτό ανά κλειδί.
- **Ημερήσιο όριο**: έως 500 κλήσεις/ημέρα ανά κλειδί.

Αν ξεπεραστεί κάποιο όριο, το εργαλείο AI θα λάβει μήνυμα σφάλματος και θα πρέπει να περιμένει πριν ξαναδοκιμάσει.

---

## Ανάκληση πρόσβασης

- Για να ανακαλέσετε ένα συγκεκριμένο κλειδί API, ή να αποσυνδέσετε μια εφαρμογή συνδεδεμένη
  μέσω OAuth (π.χ. το Claude), πατήστε το εικονίδιο διαγραφής δίπλα της στη λίστα — και τα δύο
  είδη διαχειρίζονται από το ίδιο σημείο.
- Για να απενεργοποιήσετε εντελώς το Etherly Insights, πατήστε **«Απενεργοποίηση»** — όλα τα
  ενεργά κλειδιά και οι συνδεδεμένες εφαρμογές ανακαλούνται άμεσα.

<Note>
Η ανάκληση ισχύει άμεσα· δεν υπάρχει καθυστέρηση ή cache που θα επέτρεπε σε ένα ανακληθέν κλειδί ή μια αποσυνδεδεμένη εφαρμογή να συνεχίσει να λειτουργεί.
</Note>
