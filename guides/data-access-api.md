## Τι είναι η Πρόσβαση Δεδομένων

Η Πρόσβαση Δεδομένων σάς επιτρέπει να διαβάζετε τα δεδομένα κρατήσεων και εσόδων σας
προγραμματιστικά, μέσω ενός απλού REST API με κλειδί, χωρίς να χρειάζεται κάποιο εργαλείο AI. Είναι
χρήσιμη αν θέλετε να τροφοδοτήσετε τα δικά σας dashboards, spreadsheets ή εσωτερικά εργαλεία με τα
δεδομένα σας.

<Note>
Ψάχνετε να συνδέσετε ένα εργαλείο AI (Claude, ChatGPT, Grok) ώστε να κάνετε ερωτήσεις στα φυσικά
σας δεδομένα; Δείτε τον οδηγό **«Etherly Insights»**.
</Note>

<Note>
Η Πρόσβαση Δεδομένων είναι **αυστηρά μόνο για ανάγνωση**. Καμία κλήση δεν μπορεί να δημιουργήσει,
να τροποποιήσει ή να διαγράψει δεδομένα.
</Note>

---

## Απόρρητο επισκεπτών

Τα στοιχεία επισκεπτών **δεν** επιστρέφονται ποτέ: όνομα, email, τηλέφωνο και αριθμός
ταυτοποίησης δεν εμφανίζονται ποτέ στα αποτελέσματα. Επιτρέπονται μόνο μη-προσωποποιήσιμα στοιχεία
(διανυκτερεύσεις, ποσά, κανάλι κράτησης και χώρα προέλευσης του επισκέπτη — υπολογισμένη από τον
κωδικό χώρας του τηλεφώνου, όχι πραγματική εθνικότητα).

## Ενεργοποίηση

<Steps>
  <Step title="Μεταβείτε στις ρυθμίσεις">
    Πηγαίνετε στις **Ρυθμίσεις → Για Προχωρημένους** και βρείτε την ενότητα **«Πρόσβαση
    δεδομένων (API)»**.
  </Step>
  <Step title="Ενεργοποιήστε τη λειτουργία">
    Πατήστε **«Ενεργοποίηση»**.
  </Step>
  <Step title="Δημιουργήστε ένα κλειδί API">
    Πατήστε **«Νέο key»**, δώστε έναν τίτλο (π.χ. «Εσωτερικό dashboard») και πατήστε
    **«Δημιουργία»**.

    <Warning>
    Το κλειδί εμφανίζεται **μία μόνο φορά**. Αντιγράψτε το αμέσως — δεν θα μπορείτε να το δείτε ξανά ολόκληρο.
    </Warning>
  </Step>
</Steps>

## Σύνδεση της εφαρμογής σας

Στην ίδια ενότητα θα βρείτε:

| Στοιχείο | Χρήση |
|---|---|
| **REST endpoint** (`/api/data-access/v1`) | Βάση για τις κλήσεις σας (π.χ. `/api/data-access/v1/business-overview`) |
| **OpenAPI έγγραφο** (`/api/data-access/v1/openapi.json`) | Πλήρες σχήμα για εργαλεία που παράγουν clients αυτόματα |

Χρησιμοποιήστε το κλειδί σας ως **Bearer token** στο header `Authorization`:

```bash
curl -H "Authorization: Bearer eth_data_..." \
  "https://<το-domain-σας>/api/data-access/v1/business-overview?from=2026-01-01&to=2026-01-31"
```

## Διαθέσιμα εργαλεία

Τα ίδια εργαλεία με το Etherly Insights — κάθε endpoint αντιστοιχεί σε ένα εργαλείο.

### Έσοδα & απόδοση

| Endpoint | Περιγραφή |
|---|---|
| **business-overview** | Συνολικές κρατήσεις, διανυκτερεύσεις, έσοδα και ανάλυση ανά κανάλι για ένα εύρος ημερομηνιών |
| **revenue-summary** | Σύγκριση κρατήσεων, διανυκτερεύσεων και εσόδων του τρέχοντος μήνα ή εβδομάδας με την προηγούμενη |
| **revenue-breakdown** | Ανάλυση ποσών σε μεικτά έσοδα, φόρους/τέλη, προμήθεια καναλιού και καθαρή απόδοση ιδιοκτήτη |
| **cancellation-analysis** | Ποσοστό ακυρώσεων συνολικά και ανά κανάλι κράτησης, μαζί με τα χαμένα έσοδα |

### Τιμολόγηση & πληρότητα

| Endpoint | Περιγραφή |
|---|---|
| **booking-pace** | Πόσο έχει προχωρήσει η πληρότητα μιας μελλοντικής περιόδου σε σχέση με την αντίστοιχη περυσινή |
| **occupancy** | Ποσοστό πληρότητας, μέση τιμή διανυκτέρευσης (ADR) και έσοδα ανά διαθέσιμο δωμάτιο (RevPAR) ανά property |

### Κατατάξεις & συγκρίσεις

| Endpoint | Περιγραφή |
|---|---|
| **channel-profitability** | Ποιο κανάλι κράτησης αποφέρει τα περισσότερα καθαρά έσοδα ανά διανυκτέρευση |
| **seasonality** | Κρατήσεις, διανυκτερεύσεις και έσοδα ανά μήνα για ένα ημερολογιακό έτος |
| **length-of-stay-value** | Σύγκριση μέσης τιμής διανυκτέρευσης ανάμεσα σε μεγάλες διαμονές (28+ διανυκτερεύσεις) και μικρότερες |
| **property-leaderboard** | Κατάταξη όλων των properties σας βάσει εσόδων, κρατήσεων ή διανυκτερεύσεων |

### Κρατήσεις & επισκέπτες

| Endpoint | Περιγραφή |
|---|---|
| **booking-search** | Εύρεση κράτησης βάσει κωδικού κράτησης, ή μέρους του ονόματος επισκέπτη (χωρίς εμφάνιση του ονόματος στα αποτελέσματα) |
| **checkin-completion-status** | Ποιοι επερχόμενοι επισκέπτες έχουν ολοκληρώσει το Online Check-In |
| **arrivals-departures** | Ποιοι αφικνούνται/αναχωρούν σε ένα εύρος ημερομηνιών |
| **guest-nationality-mix** | Κατανομή χώρας προέλευσης επισκεπτών για ένα εύρος ημερομηνιών |
| **list-properties** | Λίστα των properties σας (όνομα, κανάλι, τοποθεσία) |

### Λειτουργικά

| Endpoint | Περιγραφή |
|---|---|
| **operational-backlog** | Πλήθος κρατήσεων που εκκρεμούν για τιμολόγηση, ακυρωμένων, σε παύση, μεγάλης διαμονής και προς χειροκίνητο έλεγχο |
| **cleaning-readiness** | Ποια properties είναι έτοιμα για μια συγκεκριμένη ημέρα |

### Αγορά περιοχής

| Endpoint | Περιγραφή |
|---|---|
| **market-snapshot** | Εικόνα της αγοράς βραχυχρόνιας μίσθωσης στην περιοχή σας |
| **market-comparison** | Σύγκριση της δικής σας πληρότητας και μέσης τιμής με την αγορά της περιοχής για έναν μήνα |
| **property-market-position** | Πού βρίσκεται ένα κατάλυμα απέναντι στα συγκρίσιμα της περιοχής του |

<Note>
Τα εργαλεία αγοράς βασίζονται σε εξωτερική πηγή δεδομένων αγοράς και διαθέτουν ημερήσιο όριο χρήσης ανά λογαριασμό.
</Note>

## Όρια χρήσης

- **Ημερήσιο όριο**: έως 500 κλήσεις/ημέρα, συνολικά για όλα τα κλειδιά σας μαζί.
- Ισχύει επίσης όριο κλήσεων ανά λεπτό για λόγους ασφαλείας.

Αν ξεπεραστεί κάποιο όριο, θα λάβετε μήνυμα σφάλματος και θα πρέπει να περιμένετε πριν ξαναδοκιμάσετε.

---

## Ανάκληση πρόσβασης

- Για να ανακαλέσετε ένα συγκεκριμένο κλειδί API, πατήστε το εικονίδιο διαγραφής δίπλα του στη
  λίστα.
- Για να απενεργοποιήσετε εντελώς την Πρόσβαση Δεδομένων, πατήστε **«Απενεργοποίηση»** — όλα τα
  ενεργά κλειδιά ανακαλούνται άμεσα.

<Note>
Η ανάκληση ισχύει άμεσα· δεν υπάρχει καθυστέρηση ή cache που θα επέτρεπε σε ένα ανακληθέν κλειδί να συνεχίσει να λειτουργεί.
</Note>
