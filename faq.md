# Συχνές Ερωτήσεις

## Ρύθμιση & Σύνδεση

<AccordionGroup>
  <Accordion title="Οι κρατήσεις μου εμφανίζονται σε κατάσταση NEEDS_SETUP. Τι κάνω;">
    Το κατάλυμα δεν έχει πλήρη ρύθμιση. Μεταβείτε στα **Καταλύματα**, επιλέξτε το κατάλυμα και συμπληρώστε τα παρακάτω πεδία:
    - **Προεπιλεγμένη Επαφή Αποδείξεων** — λογαριασμός πελάτη στην Elorus
    - **Σειρά Τιμ.** — σειρά αρίθμησης τιμολογίων
    - **Κατηγορία** — για το Τέλος Κλιματικής Ανθεκτικότητας
    - **Οργανισμός** — το Elorus organization έκδοσης

    Μετά την **Αποθήκευση**, οι κρατήσεις αλλάζουν σε `READY` στον επόμενο συγχρονισμό.
  </Accordion>

  <Accordion title="Το Hosthub API key μου έχει αλλάξει. Πώς το ανανεώνω;">
    Μεταβείτε στις **Ρυθμίσεις** → **Διασυνδέσεις**. Βρείτε τη σύνδεση Hosthub και ενημερώστε το API key. Ο συγχρονισμός συνεχίζεται κανονικά χωρίς απώλεια δεδομένων.
  </Accordion>

  <Accordion title="Μπορώ να έχω πολλά Elorus organizations υπό τον ίδιο λογαριασμό;">
    Ναι. Μεταβείτε στις **Ρυθμίσεις** → **Διασυνδέσεις** → **«Νέα Σύνδεση»** για να προσθέσετε νέο API key Elorus. Κάθε κατάλυμα μπορεί να αντιστοιχιστεί ανεξάρτητα σε διαφορετικό **Οργανισμό** — χρήσιμο αν έχετε καταλύματα σε διαφορετικά ΑΦΜ.
  </Accordion>
</AccordionGroup>

## Αποτυχίες Αποστολής

<AccordionGroup>
  <Accordion title="Η αποστολή αποτυγχάνει με σφάλμα επαφής. Τι σημαίνει;">
    Η **Προεπιλεγμένη Επαφή Αποδείξεων** που έχει οριστεί για το κατάλυμα δεν υπάρχει ή δεν είναι ενεργή στην Elorus. Ελέγξτε στην Elorus → Επαφές αν η επαφή είναι ενεργή, και επαναλέξτε την από τα **Καταλύματα**.
  </Accordion>

  <Accordion title="Μια κράτηση είναι σε κατάσταση SENT_WITH_ERRORS. Τι γίνεται;">
    Το Παραστατικό Διαμονής και η Απόδειξη Εξόφλησης **εκδόθηκαν επιτυχώς**, αλλά το **Τέλος Κλιματικής Ανθεκτικότητας** απέτυχε. Ελέγξτε:

    1. Αν η **Κατηγορία** είναι σωστά ρυθμισμένη στα **Καταλύματα**
    2. Αν οι φορολογικοί κανόνες στην Elorus είναι ενημερωμένοι

    Επικοινωνήστε με τη support στο [support@etherly.app](mailto:support@etherly.app) αν το πρόβλημα επιμένει.
  </Accordion>

  <Accordion title="Μπορώ να ακυρώσω ένα παραστατικό που εκδόθηκε κατά λάθος;">
    Η ακύρωση παραστατικών γίνεται **απευθείας στην Elorus** — δεν υποστηρίζεται από το Etherly Sync. Μετά την ακύρωση στην Elorus, η κράτηση παραμένει σε κατάσταση `SENT`. Επικοινωνήστε με τη support για επαναφορά κατάστασης αν χρειαστεί επανέκδοση.
  </Accordion>

  <Accordion title="Η αποστολή αποτυγχάνει επανειλημμένα. Τι ελέγχω;">
    Με τη σειρά:
    1. **Elorus API key** — ελέγξτε αν είναι ακόμα έγκυρο στις **Ρυθμίσεις** → **Διασυνδέσεις**
    2. **Σειρά Τιμ.** — ελέγξτε στην Elorus αν η σειρά αρίθμησης είναι ενεργή
    3. **Οργανισμός** — βεβαιωθείτε ότι ο οργανισμός επιτρέπει τον τύπο παραστατικού
    4. **Credits** — ελέγξτε το υπόλοιπό σας στο **Credits**

    Αν κανένα από τα παραπάνω δεν λύνει το πρόβλημα, επικοινωνήστε με τη support.
  </Accordion>
</AccordionGroup>

## Συγχρονισμός

<AccordionGroup>
  <Accordion title="Γιατί δεν συγχρονίζονται νέες κρατήσεις;">
    Πιθανές αιτίες:
    - **Hosthub API key ληγμένο ή άκυρο** — ελέγξτε στις **Ρυθμίσεις** → **Διασυνδέσεις**
    - **Νέες κρατήσεις χωρίς ημερομηνία αναχώρησης** — δεν εμφανίζονται μέχρι να οριστεί ημερομηνία αναχώρησης
    - **Hosthub API timeout** — το σύστημα επαναλαμβάνει αυτόματα, αλλά αν επιμένει επικοινωνήστε με τη support

    Δοκιμάστε χειροκίνητο συγχρονισμό πατώντας **«Sync από Hosthub»** στο **Κρατήσεις**.
  </Accordion>

  <Accordion title="Μια κράτηση ακυρώθηκε στην Hosthub αλλά εξακολουθεί να εμφανίζεται. Γιατί;">
    Οι ακυρώσεις ανιχνεύονται στον επόμενο συγχρονισμό. Μόλις εντοπιστούν, η κράτηση επισημαίνεται ως ακυρωμένη και αφαιρείται από την ουρά τιμολόγησης. Αν εξακολουθεί να εμφανίζεται ενεργή, πατήστε **«Sync από Hosthub»**.
  </Accordion>
</AccordionGroup>

## Τέλος Κλιματικής Ανθεκτικότητας

<AccordionGroup>
  <Accordion title="Πώς λειτουργεί ο υπολογισμός για κρατήσεις που εκτείνονται σε δύο εποχές;">
    Αν μια κράτηση εκτείνεται μεταξύ χειμερινής (1 Νοεμβρίου – 31 Μαρτίου) και καλοκαιρινής περιόδου (1 Απριλίου – 31 Οκτωβρίου), το σύστημα:
    1. Υπολογίζει τις νύχτες ανά εποχή
    2. Εφαρμόζει τον αντίστοιχο συντελεστή για κάθε εποχή
    3. Δημιουργεί **ξεχωριστά παραστατικά** τέλους — ένα για κάθε εποχή

    Όλα γίνονται αυτόματα, χωρίς καμία ενέργεια από εσάς.
  </Accordion>

  <Accordion title="Τι γίνεται αν αλλάξουν τα ποσά του κλιματικού τέλους;">
    Τα ποσά αντλούνται από τους φορολογικούς κωδικούς της **Elorus** — δεν είναι hardcoded στο σύστημα. Αν αλλάξει η νομοθεσία, αρκεί να ενημερώσετε τα ποσά στην Elorus και το Etherly Sync τα ανακτά αυτόματα.
  </Accordion>

  <Accordion title="Δεν βρίσκω την κατηγορία του καταλύματός μου στη λίστα. Τι κάνω;">
    Οι κατηγορίες προέρχονται από τους φορολογικούς κωδικούς που έχετε ορίσει στην Elorus. Αν δεν εμφανίζεται η κατάλληλη **Κατηγορία**, βεβαιωθείτε ότι έχετε δημιουργήσει τον αντίστοιχο φορολογικό κωδικό στην Elorus. Αν είστε αβέβαιοι, συμβουλευτείτε τον λογιστή σας.
  </Accordion>
</AccordionGroup>

## Αυτόματη Τιμολόγηση

<AccordionGroup>
  <Accordion title="Η αυτόματη τιμολόγηση δεν εκτελέστηκε χθες βράδυ. Γιατί;">
    Πιθανές αιτίες:
    - Δεν υπήρχαν κρατήσεις σε κατάσταση `READY` τη συγκεκριμένη **Ώρα εκτέλεσης**
    - Τελείωσαν τα credits — ελέγξτε στο **Credits**
    - Ο λογαριασμός ήταν εκτός **Περιόδου Χάριτος**
    - Τεχνικό πρόβλημα (σπάνιο — επικοινωνήστε με τη support)
  </Accordion>

  <Accordion title="Μπορώ να αλλάξω την ώρα εκτέλεσης;">
    Ναι, ανά πάσα στιγμή. Στο **Κρατήσεις → Αυτόματη τιμολόγηση**, αλλάξτε την **Ώρα εκτέλεσης**. Η αλλαγή αποθηκεύεται αυτόματα και ισχύει από την επόμενη εκτέλεση.
  </Accordion>
</AccordionGroup>

## Χρεώσεις & Credits

<AccordionGroup>
  <Accordion title="Πότε λήγει η Δωρεάν Δοκιμαστική Περίοδος μου;">
    Η **Δωρεάν Δοκιμαστική Περίοδος** διαρκεί **30 ημέρες** από τη δημιουργία του λογαριασμού. Η ακριβής ημερομηνία λήξης εμφανίζεται στο **Credits**.
  </Accordion>

  <Accordion title="Τι γίνεται αν τελειώσουν τα credits μου;">
    Ενεργοποιείται αυτόματα η **Περίοδος Χάριτος** — η αποστολή παραστατικών συνεχίζεται κανονικά ακόμα και με 0 credits. Μετά τη λήξη της Περιόδου Χάριτος, οι νέες αποστολές μπλοκάρονται μέχρι να αγοράσετε credits από το **Credits** → **«Αγορά Credits»**.
  </Accordion>

  <Accordion title="Πώς ρυθμίζω την αυτόματη ανανέωση credits (Auto Top-up);">
    Μεταβείτε στο **Credits** → ενότητα **Auto Top-up** → **«Ενεργοποίηση Auto Top-up»**. Ορίστε **Όριο Ενεργοποίησης** (π.χ. 10 credits) και επιλέξτε ποιο πακέτο να αγοράζεται αυτόματα. Απαιτείται αποθηκευμένη κάρτα μέσω **«Προσθήκη Κάρτας»**.
  </Accordion>
</AccordionGroup>
